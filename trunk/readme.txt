== Description ==

If you need to add the post's featured image as a Open Graph meta tag to the RSS feed, ie: for Apple News.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/image-meta-tag` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. It just works!

== Frequently Asked Questions ==

Is there a settings screen?
No, it just works!

= Who is this plugin for? =

If you have your content syndicated with Apple News through RSS, and your images aren't populating, then this plugin might help you out.
